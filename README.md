Twitter Judge
by Mark Goldberg, Nathaniel Grudzinski
2016

Our CS 437 final project.
~~~~~~~

How to run this program:

1. Install Python.

On Windows, navigate to https://www.python.org/downloads/windows/
and download the latest Python 2 Release (currently 2.7.12). Install it.

On linux, python should already be installed.

2. Launch "main.py" as a python application.

On Windows, you should be able to double-click the file in explorer to run it.

On linux, navigate to the file's directory in your terminal and enter the command "python main.py".

3. Click the box at the bottom of the window next to the "Enter username:" text, type the username of the twitter user you wish to judge, and click the "Judge!" button to the right.
